In the following questions:
Answer is 1 if only conclusion (a) necessarily follows.
Answer is 2 if only conclusion (b) necessarily follows.
Answer is 3 if both conclusion (a) and conclusion (b) necessarily follows.
Answer is 4  if neither conclusion (a) nor conclusion (b) necessarily follows.
