Creating a Dll And Using It With C#


Here is a very quick and easy way to create a Dll and use it in C#.


NameSpace.CS

//Compilation is done using "csc /target:library NameSpace.cs "

//UseName.cs is the file which uses this dll.

namespace DllMessage
{
public class HiMessage
{
public string Messages()
{
return "Hi, Thats Mine";
}
}
}


UserName.CS

// Use this Command to compile "csc /reference:NameSpace.dll UseName.cs"

using DllMessage;

public class GetHi
{
public static void Main()
{
HiMessage hi = new HiMessage();
System.Console.WriteLine(hi.Messages());
}
}
